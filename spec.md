# Spec — ROMEL Phone App

**Confirmed scope:** Single mobile viewport prototype, HTML/CSS, GitHub Pages deploy.
**Stack:** Plain HTML5 + inline CSS; mobile-first; responsive scaling.
**Files:**
- `index.html` — full prototype
- `design/design.md` — design rationale and tokens
- `brief.md`, `spec.md`, `log.md` — project docs
**Acceptance criteria:**
- [x] Brand file read and applied.
- [x] Mobile viewport renders cleanly at 375px and scales up.
- [x] Agent roster uses exact roles from brand; Dr. Bayle shown as "—".
- [x] Steel blue used for borders/focus only; no solid accent fills.
- [x] One primary CTA: "Request access".
- [x] Live URL returns HTTP 200.
